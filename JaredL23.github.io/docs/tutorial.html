Tutorial: First API Call

  1. Start the API (see Getting Started).
  2. Create a claim:
curl -X POST http://127.0.0.1:8000/claims \
-H "Content-Type: application/json" \
-d '{"member_id":"ABC123", "amount":100.5, "status":"submitted"}'

Copy the id and fetch it:

curl http://127.0.0.1:8000/claims/{id}
